# linux-aio

Personal aio script for quick setup on Debian/Ubuntu-based distros and Fedora

## Current features
- tmux + TPM setup
- zsh + Oh My Zsh + custom setup
- dry run `--dry-run`

## Prerequisites
- bash
- sudo
- wget/curl

## Installation

### curl

```

bash <(curl -fsSL https://raw.githubusercontent.com/diluccio42/linux-aio/master/install.sh) --all

```

### wget

```

bash <(wget -qO- https://raw.githubusercontent.com/diluccio42/linux-aio/master/install.sh) --all

```
